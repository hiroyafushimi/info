COHÉRENCE — ZEN × NEUROSCIENCE

1. Informations collectées

Les informations collectées par l'Application sont strictement limitées au minimum nécessaire.

Informations de compte

  • Firebase UID (identifiant unique d'authentification)
  • Nom d'affichage (facultatif, obtenu via Apple ID ou compte Google)

Informations non collectées

  • Adresse e-mail
  • Numéro de téléphone
  • Données de localisation
  • Contacts
  • Photos ou médias
  • Identifiant publicitaire (IDFA)

2. Données du journal quotidien

Le contenu saisi dans le journal quotidien est stocké exclusivement sur votre appareil. Il n'est jamais transmis à nos serveurs.

  • Stockage : Base de données locale sur l'appareil (SQLite)
  • Sauvegarde : Apple iCloud (uniquement si activé dans les réglages iCloud de votre appareil, protégé par le chiffrement d'Apple)
  • Transmission serveur : Aucune. Les données du journal ne sont jamais envoyées à nos serveurs

3. Données de session

L'historique des sessions de méditation (date, durée, étape, humeur) est conservé aux fins suivantes :

  • Sur l'appareil : Affichage des séries, tableau de bord de progression, calcul des tendances d'humeur
  • Synchronisation cloud : Stockage chiffré sur Firebase Firestore (pour la continuité des séries entre appareils). Les utilisateurs anonymes ne font l'objet d'aucune synchronisation cloud

4. Analyses d'utilisation

L'Application utilise PostHog Cloud EU pour l'analyse d'utilisation.

  • Centre de données : Union européenne
  • Utilisation de cookies : Aucune (mode sans cookies)
  • Données collectées : Événements tels que démarrage/fin de session, navigation entre écrans, utilisation des fonctionnalités. Un identifiant interne (Firebase UID) est utilisé pour relier les événements, mais aucune donnée nominative (nom, adresse e-mail) n'est transmise

Firebase Analytics et Google Analytics ne sont pas utilisés.

5. Abonnement

La gestion des abonnements est assurée par RevenueCat. Les achats sont traités via l'App Store d'Apple. L'Application ne traite directement aucune information de paiement telle que les numéros de carte bancaire.

6. Partage de données avec des tiers

L'Application ne vend, ne loue ni ne partage vos données personnelles avec des tiers. Les données sont partagées uniquement avec les prestataires de services suivants :

  • Firebase Authentication : Authentification des utilisateurs (Google LLC, États-Unis)
  • Firebase Firestore : Synchronisation de l'historique des sessions (Google LLC, États-Unis)
  • PostHog Cloud EU : Analyse d'utilisation (PostHog Inc., centre de données UE)
  • RevenueCat : Gestion des abonnements (RevenueCat Inc., États-Unis)
  • Apple iCloud : Sauvegarde du journal (Apple Inc. — selon les réglages de l'utilisateur)

Transferts internationaux : Certaines données sont transférées vers les États-Unis (Firebase, RevenueCat). Ces transferts sont encadrés par les clauses contractuelles types de la Commission européenne et les politiques de protection des données de chaque prestataire.

7. Conservation et suppression des données

Vous pouvez supprimer vos données à tout moment.

  • Suppression du compte : Accessible depuis « Réglages » → « Supprimer le compte » dans l'application. Cette action supprime les données d'authentification Firebase, l'historique des sessions sur Firestore et les données client RevenueCat
  • Données du journal : La suppression de l'application de votre appareil efface les données du journal stockées localement. Pour la sauvegarde iCloud, supprimez-la manuellement depuis les réglages iCloud de votre appareil
  • Données d'analyse : Les données envoyées à PostHog sont liées à un identifiant interne. Après suppression du compte, le lien avec votre identité est rompu

8. Confidentialité des enfants

L'Application ne s'adresse pas aux enfants de moins de 13 ans et ne collecte pas intentionnellement de données les concernant.

9. Sécurité

Les mesures suivantes sont mises en œuvre pour protéger vos données :

  • Toutes les communications avec Firebase sont protégées par chiffrement TLS
  • Les données du journal sont stockées uniquement sur l'appareil et ne sont jamais transmises à nos serveurs
  • Les sauvegardes iCloud sont protégées par le chiffrement d'Apple
  • Aucun suivi publicitaire ni collecte d'IDFA n'est effectué

10. Règlement général sur la protection des données (RGPD)

Si vous résidez dans l'Union européenne, vous disposez des droits suivants :

  • Droit d'accès à vos données
  • Droit de rectification
  • Droit à l'effacement (« droit à l'oubli »)
  • Droit à la portabilité des données
  • Droit d'opposition au traitement
  • Droit d'introduire une réclamation auprès d'une autorité de contrôle (CNIL en France)

Pour exercer ces droits, veuillez nous contacter à l'adresse indiquée ci-dessous.

Base juridique du traitement : Le traitement des données est fondé sur votre consentement (article 6(1)(a) du RGPD) et sur l'exécution du contrat de service (article 6(1)(b) du RGPD).

Responsable du traitement : Hiroya Fushimi, entrepreneur individuel opérant sous le nom Cohérence, basé au Japon.

11. Loi japonaise sur la protection des informations personnelles

L'Application est conforme à la loi japonaise sur la protection des informations personnelles (個人情報保護法) :

  • Les finalités d'utilisation sont explicitement décrites dans cette politique
  • Le traitement des informations personnelles est limité au strict nécessaire
  • Les demandes de divulgation, rectification et suppression sont traitées
  • Les destinataires tiers des données personnelles sont identifiés
  • Les pays de destination des transferts internationaux sont indiqués
